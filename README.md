# AI Visual Novel Game

A minimal interactive visual novel built with Kaboom.js. Click on NPCs to talk to them in a fantasy village setting.

## Play

Visit: **https://chfr19820610-cell.github.io/ai-vn-game/**

## Characters

- **Elder Kael** - The village elder who speaks of dark forces
- **Captain Mira** - Leader of the village guard
- **Luna the Mystic** - A seer who reads the stars

## Tech

Built with [Kaboom.js](https://kaboomjs.com/) — a JavaScript game programming library.

## Local Development

Open `index.html` in any browser. No build step required.
